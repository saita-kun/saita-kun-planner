# scoring-strategy

この書き方メモは、小規模事業者持続化補助金 第20回公募の同梱 spec から作った provider note です。制度事実・数値・要件の根拠は必ず spec の clause を見ます。clause 引用のない助言は一般知見であり、数値・要件の根拠には使いません。

## 枠・特例の考え方

- 基本の補助率と補助上限は spec の funding を正とし、本文では選んだ特例の要件を満たす証憑まで確認します。[clause: funding-001]
- インボイス特例は、補助事業終了時点で適格請求書発行事業者の登録を受け、免税事業者歴または創業時期の要件を満たす場合に限ります。[clause: funding-005]
- 賃金引き上げ特例は、補助事業実施期限日を終点とした連続する12か月と前年同月の12か月を比較し、従業員1人あたり給与支給総額を3.0%以上増加させる要件です。[clause: funding-006]
- 赤字事業者の補助率3/4は、賃金引き上げ特例に取り組む事業者のうち、直近1期または直近1年間の課税所得金額がゼロ以下であることが根拠です。[clause: funding-007]

## 加点の扱い

- 賃金引き上げ加点は、2027年4月1日から補助事業実施期限日までの期間と前年同月の12か月を比べ、従業員1人あたり給与支給総額が年平均2.0%以上増加した事業者が対象です。[clause: bonus-001]
- 赤字賃上げ加点は、賃金引き上げ特例に申請する事業者のうち赤字である事業者が対象です。[clause: bonus-002]
- 事業環境変化加点は、ウクライナ情勢、原油価格、LPガス価格等の高騰、米国による相互関税の影響を受けている事業者が対象です。[clause: bonus-003]
- 経営力向上計画加点は、各受付締切回の基準日までに認定を受けている事業者が対象で、認定書の写し提出が必要です。[clause: bonus-011]
- 事業継続力強化計画策定加点は、申請受付締切日までに認定を受け、実施期間が終了していない認定事業者が対象です。[clause: bonus-008]

## 申請本文への反映

- 加点や特例は、該当する場合だけ本文・添付・入力項目へ反映します。未確認の認定、証明書、賃金実績、被災証明、計画公表状況は本文で断定せず、顧客本人の資料確認に回します。
- 加点を狙う場合でも、中心は経営計画・補助事業計画の審査観点です。自社分析、経営方針、補助事業計画の有効性、積算の透明性を本文の主軸にします。[clause: review-002]
