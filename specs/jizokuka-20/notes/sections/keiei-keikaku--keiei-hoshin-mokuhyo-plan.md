# 経営計画：経営方針・目標と今後のプラン

この section-note は、顧客本人が経営方針と補助事業の接続を整理するための書き方メモです。制度事実・数値・要件は spec の clause を根拠にし、目標数値は顧客資料で確認します。

## 書く観点

- 経営方針・目標と今後のプランの適切性が審査観点です。現状分析、市場ニーズ、強み・弱みから、なぜ今この補助事業に取り組むのかを一本につなげます。[clause: review-002]
- 補助事業は、販路開拓等または販路開拓等とあわせて行う業務効率化・生産性向上の取組である必要があります。方針にもこの目的を反映します。[clause: target-business-001]
- 売上高・売上総利益の増加を見込める事業であることが求められるため、目標は顧客本人の根拠資料で確認してから置きます。[clause: target-business-004]

## 避けること

- 目標だけを大きく書かず、実行手段、体制、費用、スケジュールと整合させます。[clause: review-002]
