# 01. GitHub アカウントの作成

このキットは GitHub というサービス上で配布されています。まず無料のアカウントを作ります。

## 手順

1. ブラウザで <https://github.com/signup> を開きます。
2. メールアドレスを入力します（普段使うものでかまいません）。
3. パスワードを設定します（推測されにくいものに）。
4. ユーザー名（半角英数字）を決めます。これがあなたの GitHub の名前になります。
5. 画面の案内に従って、メールに届いた確認コードを入力します。
6. 「個人用（Free）」プランを選びます。有料プランは不要です。

これでアカウント作成は完了です。

## なぜ private（非公開）repo を使うのか

このキットでは、あなたの作業 repo を **private（自分だけが見られる状態）** で作ります。理由は、会社情報など機密を扱うためです。`input/` フォルダは git 管理外ですが、作業 repo 自体も private にしておくのが安全です。

> 旧 collaborator 招待モデルは撤回済みです。提供側をあなたの private repo に共同作業者として招待することは利用条件ではありません。現在のデータ提供は result-report の任意提出であり、提出しなくても中核機能は使えます。何を送るか・送らないかは `docs/collaborator-招待手順.md`（現行の result-report 任意提出ガイド）と `TERMS.md` で確認できます。
