# はじめに（オンボーディング全体地図）

このキットは、補助金申請用の事業計画書の叩き台を、**あなた自身が** Claude Code を使って作るための作業環境です。提供側は申請書類に一切触れません（作成者はあなた本人）。

## あなたは今どの段階ですか？

下のどれに当てはまるかで、読む順番が変わります。

1. **GitHub アカウントを持っていない** → `01-githubアカウント作成.md` から始めてください。
2. **GitHub はあるが Claude Code を使っていない** → `02-claude-codeセットアップ.md` へ。
3. **GitHub も Claude Code もある** → `03-このキットを自分のものにする.md` へ。
4. **すでにこのキットを自分の repo にして Claude Code で開いた** → Claude Code で `/setup` と入力してください。あとは Claude Code が案内します。

## 全体の流れ

```
GitHub アカウント作成（01）
   → Claude Code セットアップ（02）
   → このキットを「Use this template」で自分の private repo にして clone・Claude Code で開く（03）
   → /setup（環境チェック・利用規約の同意・result-report 任意提出の確認）
   → /start
   → 入口A: /select-subsidy または 入口B: /ingest-guidelines
   → /confirm-spec → /build-pack
   → /intake → /subsidy-fit → /plan-deliverables → /draft-section → /review → /verify → /finalize
```

## 前提・注意

- 本キットは **Claude Code を使えるクライアント向け**です。
- あなたの会社情報など機密は `input/` フォルダに置きます。ここは git 管理外なので、提供側にも公開リポにも出ません。
- 数値や要件は推測しません。公式の募集要項とあなたの資料が根拠です（詳細 → `docs/法務とスコープ.md`）。
- 利用条件・データの扱いは `TERMS.md` / `docs/data-policy.md` を参照してください。
