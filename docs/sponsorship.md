# 法人スポンサー案内

> **独立性宣言**: スポンサー資金は推薦順位・診断結果に一切影響しません。スポンサーシップは、補助金フィード採用、診断ロジック、重み付け、検証ゲート、利用者への助言内容を購入する手段ではありません。data-charter の COI ファイアウォールは [docs/governance/data-charter.md](governance/data-charter.md) §7 を参照してください。

saita-kun-planner は、補助金申請者が自分の Claude Code と公式募集要項を使い、事業計画書の叩き台をセルフサーブで作るための OSS ハーネスです。法人スポンサーに支えてほしいのは、個別の申請支援ではなく、誰でも検証できるデータ維持インフラです。

## スポンサーが支えるもの

スポンサー資金は、補助金情報を読むための公共財を維持するために使います。

- 公募要領の改訂追随と、古くなった spec の検知。
- confirmed spec、補助金パック、書き方メモを含む spec の鮮度維持。
- schema、検証ツール、manual、FAQ の継続的な改善。
- 利用者が自分で募集要項と照合できる手順、テスト、サンプルの整備。

補助金データは、無料でも鮮度を失うと有害になり得ます。スポンサーは、推薦順位を買うのではなく、制度情報を読み解くための共通基盤を支える立場です。

## L1 年間 tier（目安）

金額は初期の目安です。個別契約時には、独立性宣言、個票非提供、推薦・診断への非影響を前提に確認します。

| 区分 | 年額目安 | 位置づけ |
| --- | --- | --- |
| L1 Supporter | 25万円 | 継続的な spec 鮮度維持を支える小口協賛 |
| L1 Partner | 100万円 | 公開ハーネス、検証ツール、主要 docs の改善を支える協賛 |
| L1 Sustainer | 200万円 | 公募要領改訂追随と補助金パック拡充を支える中核協賛 |

この tier は、診断結果、推薦順位、補助金パックの採用可否、特定スポンサーへの有利な表示を約束するものではありません。

## 返礼の範囲

返礼は、名前・ロゴ掲載などの非機能的便益に限定します。

- sponsor 一覧への法人名掲載。
- ロゴ掲載。
- 公共財維持への謝辞。
- 支援趣旨を説明する短い紹介文の掲載。

次の内容は返礼に含めません。

- 補助金の推薦順位、診断結果、適合度判定への影響。
- スポンサーに有利な重み付け、表示順、検証ゲート変更。
- 個別申請支援、申請書作成代行、代理提出。
- 利用者の `input/`、申請本文、個別 result-report、生ログ、未公開資料の提供。
- 公式募集要項に基づかない補助率、補助上限、採択可能性などの断定。

作成者は顧客本人です。AI は補助、壁打ち、整理役であり、行政書士法に抵触する申請代行や代理提出は行いません。

## 透明性

スポンサー一覧、金額規模、資金使途、COI に関わる扱いは、公開できる範囲で明示します。スポンサーが関係する補助金情報を扱う場合も、同一の公開ソース基準、確認基準、レビュー基準を適用します。

スポンサーは saita-kun-planner の公共財を支える存在であり、利用者の診断結果や推薦順位を左右する存在ではありません。
