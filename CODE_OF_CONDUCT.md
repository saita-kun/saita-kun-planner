# Contributor Covenant Code of Conduct

## 私たちの誓約

私たちは、メンバー、コントリビュータ、リーダーとして、年齢、体格、見える障害または見えない障害、民族的特徴、性自認と性表現、経験年数、教育、社会経済的地位、国籍、外見、人種、宗教、性的指向その他の属性にかかわらず、誰にとってもハラスメントのない参加体験を提供することを誓います。

私たちは、開かれ、歓迎され、多様で、包摂的で、健全なコミュニティに貢献する方法で行動し、交流することを誓います。

## 私たちの基準

前向きな環境に貢献する行動の例は次のとおりです。

- 他者への共感と親切を示すこと
- 異なる意見、視点、経験を尊重すること
- 建設的なフィードバックを与え、丁寧に受け止めること
- 自分の誤りを認め、影響を受けた人へ謝意や改善を示し、経験から学ぶこと
- 個人としての自分だけでなく、コミュニティ全体にとって最善となることを重視すること

許容されない行動の例は次のとおりです。

- 性的な表現や画像の使用、性的な注意や接近
- 侮辱、軽蔑的または攻撃的なコメント、個人攻撃または政治的攻撃、荒らし行為
- 公開または非公開のハラスメント
- 明示的な許可なく、住所、メールアドレス、個人情報など他者の個人情報を公開すること
- 職業上の場において不適切と合理的に判断されるその他の行為

## 執行責任

コミュニティリーダーは、許容される行動の基準を明確にし、執行する責任があります。コミュニティリーダーは、不適切、脅迫的、攻撃的、または有害と判断したコメント、コミット、コード、wiki 編集、issue、その他の貢献を削除、編集、拒否する権利と責任を持ちます。

コミュニティリーダーは、この行動規範に反する行為をした人に対し、一時的または恒久的な参加制限を行うことがあります。

## 適用範囲

この行動規範は、すべてのコミュニティ空間に適用されます。また、個人が公の場でコミュニティを代表している場合にも適用されます。コミュニティを代表する例には、公式メールアドレスの使用、公式 SNS アカウントでの投稿、オンラインまたはオフラインイベントで任命された代表者として行動することが含まれます。

このリポジトリでは、GitHub issue、Pull Request、discussion、commit、review、メールでの連絡、関連する公開・非公開のやり取りが対象です。

## 執行

不適切、ハラスメント、またはその他の許容されない行動は、`info@subsidy-support.tech` に報告してください。すべての報告は速やかかつ公平に確認されます。

コミュニティリーダーは、報告者のプライバシーと安全を尊重します。必要な範囲を超えて、報告者、被害を受けた人、関係者の個人情報を公開しません。

## 執行ガイドライン

コミュニティリーダーは、この行動規範に違反すると判断した行為に対し、次の影響度ガイドラインに沿って対応します。

### 1. 是正

影響: 不適切な言葉づかい、またはコミュニティ内で専門的でない、歓迎的でないと判断される行動。

対応: コミュニティリーダーから、違反の内容、行動が不適切である理由、今後期待される行動を明確に伝えます。公開の謝罪を求める場合があります。

### 2. 警告

影響: 単発または連続した違反。

対応: 違反を続けた場合の結果を明示した警告を行います。一定期間、関係者との直接交流を控えるよう求める場合があります。これには、コミュニティ空間や外部チャネルでのやり取りも含まれます。条件に違反した場合、一時的または恒久的な参加制限につながることがあります。

### 3. 一時的な参加制限

影響: 不適切な行動が継続している、または重大な違反があった。

対応: 一定期間、コミュニティとの交流や公開コミュニケーションへの参加を制限します。その期間中は、対象者との直接または間接のやり取り、コミュニティ空間への投稿、メンテナーとの非公開連絡を認めない場合があります。条件に違反した場合、恒久的な参加制限につながることがあります。

### 4. 恒久的な参加制限

影響: ハラスメント、個人への脅迫、危険な行動、またはコミュニティ基準への継続的な違反を含む、一連の違反。

対応: コミュニティへの参加を恒久的に制限します。
